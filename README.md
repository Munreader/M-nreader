## Deployment Instructions

### GitHub Pages Deployment (Recommended)

1. **Repository Setup:**
   - Repository: https://github.com/Munreader/M-nreader
   - Branch: main
   - Enable GitHub Pages in Settings

2. **Custom Domain Setup:**
   - GitHub Pages settings → Custom domain
   - Add: www.munreader.com
   - Update DNS records to GitHub Pages

### Netlify Deployment (Alternative)

1. **Manual Drag-and-Drop:**
   - Go to https://app.netlify.com/drop
   - Drag all files from this folder
   - Get instant URL: random-name.netlify.app

2. **Custom Domain Setup:**
   - In Netlify dashboard: Site settings → Domain management
   - Add custom domain: munreader.com
   - Update DNS nameservers to Netlify
